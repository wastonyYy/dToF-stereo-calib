# Marker-Based Multi-Sensor Calibration

This repository will contain the official implementation and calibration datasets for the paper:

**"Robust Marker-Based Intrinsic and Extrinsic Calibration for Non-Overlapping Multi-dToF and Stereo Sensor Systems"**

## Status: Under Review ⏳

Currently, the manuscript is under peer review. To ensure the integrity of the double-blind review process (if applicable) and pending final publication decisions, the full source code and datasets are temporarily withheld. 

All materials, including:
- Targetless intrinsic calibration scripts
- Geometry-driven extrinsic calibration framework
- Sample dToF and stereo camera datasets

will be made **fully open-source and publicly available** immediately upon the acceptance of the paper.

Please check back later or contact the corresponding author for any urgent inquiries.
